Инстукция по установке сайта из резервного архива
==============================================

1. Поместите архив в пустую папку на вашем комьютере.
2. Извлеките файл install.php находящийся в архиве и разместите рядом с файлом архива.
3. Скорируйте файл архива и файл install.php на ваш WEB сервер.
4. Откройте страницу сайта по ссылке http://АДРЕС_ВАШЕГО_САЙТА/install.php
5. Откроется страница с предложением ввести пароль для продолжения восстановления.
6. Введите пароль, если пароль верный сайт восстановитсяно, дополнительно запросятся данные о настройках базы данных.
