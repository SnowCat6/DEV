Инстукция по установке сайта из резервного архива
==============================================

1. Поместите архив в пустую папку на вашем комьютере.
2. Откройте архив в любой программе для работы с архивами.
3. Скопируйте файл index.php находящийся в архиве и разместите го рядом с файлом архива.
4. Скорируйте файл архива и файл index.php на WEB сервер где вы желаете запустить сайт.
5. Откройте страницу сайта по ссылке http://АДРЕС_ВАШЕГО_САЙТА/index.php?URL=НАЗВАНИЕ_АРХИВА (вместо НАЗВАНИЕ.zip напишите НАЗВАНИЕ.htm)
6. Откроется страница с предложением ввести пароль для продолжения восстановления.
7. Введите пароль, если пароль верный сайт восстановится и возможно, что дополнительно запросятся данные о настройках базы данных.
