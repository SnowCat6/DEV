Для работы сайта необюходимо:
1. PHP верии от 4.2 с установленным драйвером MySQLi и GD2
2. MySQL любой версии.

Установка сайта
1. Скопируйте архив в корневую папку WEB сервера.
2. Присвойте права на выполнение index.php
3. Выполните команду:
http://адрес_сайта/?URL=backup_2014-07-22-13-27.htm
4. В поле пароля введите BEING, пароль я экстренного восстановления сайта.
5. Нажмите галочку "Восстановить сайт, все текущие данные будут уничтожены"
6. Нажмите конпку "Восстановить"
7. В появившиеся поля настройки базы данных введите корректные данные вашей базу MySQL
8. Нажмите конпку "Восстановить"

Если все пройдет как задумано, сайт установится и заработает.

Для редактирвоания сайта зайдите на строаницу http://имя_сайта/login.htm
Введите логин admin без пароля.
Дажее зайдите на любую страницу, которую хотите редактировать и нажмите "Именить" в меню.
Вы можете получить больше инфорации о редактирвании наведя мышку на верхнюю строку "Панель администрирования".
